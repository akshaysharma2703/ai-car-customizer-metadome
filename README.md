# AI Car Customizer (Unity)

This project is an interactive automotive configurator developed using Unity.

## Features
- Change car body colors
- Wheel customization
- Theme-based AI recommendations (Sporty, Luxury, Minimal)
- Mouse orbit camera
- Zoom using scroll wheel
- Garage lighting and HDRI reflections

## Controls
Left Mouse Drag – Rotate camera  
Mouse Scroll – Zoom  
UI Buttons – Change color and apply theme

## How to Run
1. Download the build from the Build folder
2. Run the .exe file
3. Interact with the UI to customize the car

## Technologies Used
- Unity 6
- C#
- URP Lighting
